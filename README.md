# Parser
Abstract PHP implementation of Selenium and CURL parsers using Facebook WebDriver.

## Elements

### Driver (DriverInterface)
 - *Selenium*
 - *CURL*

### Node (NodeInterface)
 - *SeleniumNode*
 - *CurlNode*

### Search
 - *Search*
 - *Page*
 - *Result*

### Source (SourceInterface)
 - Parse index and listing
